# Cluedo

This application was created for CS670 Artificial Intelligence at NJIT.

Repository can be cloned on Github. In order to run code navigate to directory in VSCode. Within the terminal run the main.py file. 

Cluedo is a game that involves players finding out the combination of cards responsible for a murder. Players roll dice and make a suggestion based on what location they move to. Once enough information is gained, players can make an accusation of which character, weapon, and location is responsible for the murder. If one of the players has the cards, it will say that the player disproves the suggestion. If the wrong accusation is made, the player is unable to take turns for the rest of the game. The game ends when a player makes a right accusation or when all players guess wrong.
